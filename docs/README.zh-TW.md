## 🚀 Quick Start (pip)

透過 pip 安裝 Chibi 並作為命令列應用程式執行：

```bash
# 安裝軟體套件
pip install chibi

# 設定代理（新增 API 金鑰、更新設定等）
chibi config

# 啟動機器人
chibi start
```

機器人將以背景服務運行。使用 CLI 命令進行管理。

### CLI 命令

| 命令 | 描述 |
|---------|-------------|
| `chibi start` | 啟動機器人作為背景服務 |
| `chibi stop` | 停止執行中的機器人 |
| `chibi restart` | 重新啟動機器人 |
| `chibi config` | 產生或編輯組態 |
| `chibi logs` | 查看機器人日誌 |
